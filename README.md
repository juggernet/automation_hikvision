# Automation Camaras Hikvision

Proyecto de automatización de cámaras Hikvision usando Python e ISAPI.

## Funciones

- Inventario automático
- Exportación a Excel
- Obtención de:
  - IP
  - Canal
  - Modelo
  - Serie
  - Nombre NVR
  - Nombre cámara
- Cambio automático de nombres

## Tecnologías

- Python
- Requests
- OpenPyXL
- Hikvision ISAPI
